### Reconciliation Methodology

**The ChatFin 4-Step Process:**

**Step 1: Data Extraction**
- Pull GL balance (period-end)
- Extract sub-ledger detail
- Gather external sources (bank statements)
- Collect supporting documentation

**Automation:** 100% automated, runs overnight

**Step 2: Intelligent Matching**

ChatFin applies sophisticated matching logic:

**Exact Match:**
- Same amount, date, reference
- Immediate auto-match
- 60-70% of transactions

**Fuzzy Match:**
- Similar amounts (within tolerance)
- Date variance (± few days)
- Reference number patterns
- 20-25% of transactions

**Predictive Match:**
- AI learns patterns
- Vendor/customer behavior
- Historical matching rules
- 10-15% of transactions

**Total Auto-Match:** 90-95%

**Step 3: Exception Investigation**

For unmatched items, ChatFin:
- Searches for related transactions
- Identifies probable matches
- Proposes reconciling items
- Generates research notes

**Controller Action:** Review and approve (vs. research from scratch)

**Step 4: Documentation**

Auto-generated reconciliation includes:
- Beginning balance
- GL activity
- Sub-ledger activity
- Reconciling items
- Ending balance
- Variance explanation
- Supporting documents
- Sign-off workflow

**Audit-ready documentation** in seconds, not hours
