# ChatFin Security & Compliance

Enterprise-grade security for your financial data.

## Security Certifications

**SOC 2 Type II**  
Independently audited controls for:
- Security
- Availability
- Processing integrity
- Confidentiality
- Privacy

**Additional Certifications**
- ISO 27001 (in progress)
- GDPR compliant
- CCPA compliant
- PCI DSS compliant (for payment processing)

## Data Protection

**Encryption**
- **At rest:** AES-256 encryption
- **In transit:** TLS 1.3
- **Key management:** AWS KMS with rotation

**Access Controls**
- Role-based access control (RBAC)
- Multi-factor authentication (MFA)
- Single sign-on (SSO) support
- Session timeout policies
- IP whitelisting available

**Data Residency**
- US, EU, or custom region options
- No cross-border data transfer without consent
- Customer data isolation

## Infrastructure Security

**Cloud Infrastructure**
- AWS/Azure/GCP availability
- Multi-region redundancy
- 99.99% uptime SLA
- Automated failover

**Network Security**
- Web application firewall (WAF)
- DDoS protection
- Intrusion detection/prevention
- Regular penetration testing

**Monitoring**
- 24/7 security operations center
- Automated threat detection
- Real-time alerting
- Incident response procedures
