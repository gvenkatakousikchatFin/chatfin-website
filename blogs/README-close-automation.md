## The Anatomy of a Traditional Close

### Day 1-3: Data Collection
**Manual Activities:**
- Export data from multiple systems
- Import into Excel spreadsheets
- Validate data completeness
- Request missing information
- Chase down sub-ledger reports

**Problems:**
- Time-consuming exports
- Data accuracy issues
- Missing information delays
- Version control nightmares

### Day 4-6: Reconciliations
**Manual Activities:**
- Bank reconciliations (350+ accounts)
- Sub-ledger to GL reconciliation
- Intercompany reconciliations
- Balance sheet account analysis
- Investigate discrepancies

**Problems:**
- Repetitive manual matching
- High error rates (5-8%)
- Time-intensive exception research
- Documentation challenges

### Day 7-8: Accruals & Adjustments
**Manual Activities:**
- Calculate accruals (payroll, revenue, expenses)
- Prepare journal entries
- Obtain approvals
- Post to system
- Recalculate financials

**Problems:**
- Formula errors in spreadsheets
- Approval bottlenecks
- Posting mistakes
- Multiple iterations

### Day 9-10: Variance Analysis & Reporting
**Manual Activities:**
- Prepare variance schedules
- Write commentary
- Generate financial statements
- Format board presentations
- Management review

**Problems:**
- Last-minute surprises
- Limited analysis time
- Rushed commentary
- Formatting issues

**Total:** 10 days, 60-80 hours per FTE, high stress
